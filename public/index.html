WASPS. vBeta.

World And System Proceduraliser Sim.
- A project by yanfu

building on three.js and simplex noise, motivated by the recent renewed efforts in our endeavours to go beyond Earth, this project is a small idea of mine to artificially create an entire, discoverable planetary system in a web browser.

the core functions are there, that being procedural generation of a system with full customisation and an option to go in a spaceship and explore. i dont think i will make it a full game, because 1. theres already so many games out there and 2. skill issues. but i do hope there's enough detail here to hopefully spark some imagination and steer some ppl towards space. this is only a small snapshot, maybe not even a snapshot, of space exploration.
i hope y'all love it, and have fun!
